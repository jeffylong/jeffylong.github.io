前言

说到为什么会突然想起学习Python呢？前段时间听她提到爬虫，粗略知道爬虫主要用的语言是Python；其次，在某公众号看到说Python是现在排名前5的编程语言中唯一使用人数在增加的。于是，有心学习。

Python简介

Python是著名的Guido van Rossum在1989年圣诞节期间，为了打发无聊的圣诞节而编写的一个编程语言。

优点

Python是一门高级编程语言，内置了完善的基础代码库，覆盖了网络、文件、GUI、数据库、文本等大量内容，还有大量的第三方库。许多大型网站就是用Python开发的，例如YouTube、Instagram，还有国内的豆瓣。很多大公司，包括Google、Yahoo等，甚至NASA（美国航空航天局）都大量地使用Python。
Python的哲学就是“优雅、明确、简单”，尽量写容易看明白的代码，尽量写少的代码。

缺点

  1. 运行速度慢
    python是解释型语言，代码在执行时会一行一行地翻译成CPU能理解的机器码，这非常耗时。而c语言则是运行前直接编译成CPU能执行的机器码。
  2. 代码无法加密
    如果要发布你的Python程序，实际上就是发布源代码，这一点跟C语言不同，C语言不用发布源代码，只需要发布编译后的机器码，要从机器码反编译出C代码是不可能的，所以，凡是编译型的语言，都没有这个问题，而解释型的语言，则必须把源码发布出去。

安装成功后，在命令行中输入python，即可进入Python交互式环境中（>>>即表示当前模式为Python环境）

Python解释器

Python的代码文件是一个以.py为扩展名的文本文件，需要Python解释器去执行.py文件。主流的解释器有CPython、IPython、PyPy、IronPython等。
但使用最广泛的还是CPython，这是官方版本的解释器。基于c语言开发，在命令行下运行python就是启动CPython解释器，以>>>作为提示符。

开始写Python

hello,python

在文本编辑器中新建一个文件输入print('hello,python')，保存为hello.py文件（文件名只能是英文字母、数字、下划线的组合）。打开命令行窗口，切换到.py文件的目录，输入python hello.py
文本编辑器推荐Sublime Text；绝对不能用word或者记事本，因为word保存的不是纯文本文件，而记事本会自作聪明地在文件开始位置加上几个特殊字符（UTF-8 BOM），结果导致程序运行出现莫名其妙的错误。

tips:windows系统的命令行下，f:切换到F盘，dir可以查看文件列表，再cd py进入F:\py>目录。

字符串和编码

因为计算机只能处理数字，如果要处理文字，就必须先把文本转换成数字。最早的计算机在设计时采用8个比特（bit）作为一个字节（byte），所以，一个字节能表示的最大整数是255（二进制11111111=十进制255），那么要表示更大的整数需要更多的字节，比如，两个字节表示的最大整数是65535，四个字节是4294967295。
计算机最初只有127个字符被编码，即大小写英文字母、数字和一些符号，这个编码称为ASCII编码，比如A的编码是65，小写z是122。

处理中文需要至少两个字节，而且不能与ASCII编码重复，所以中国制定了GB2312编码用来编码中文字符。如法炮制，日文的编码规则是Shift_JIS,韩文的编码是Euc_kr。各国标准不同导致多语言文本中显示乱码。
因此，Unicode应运而生。它将所有语言统一到一套编码中，解决了乱码的问题。
ASCII与Unicode编码的区别在于：前者编码是1个字节，而后者通常是2个字节。
字母A用ASCII编码是十进制的65，二进制的01000001；
字符0用ASCII编码是十进制的48，二进制的00110000（注意字符0和整数0是不同的）；
汉字中已经超出了ASCII的范围，用Unicode编码是十进制的20013，二进制的01001110 00101101。
如果把ASCII编码的A用Unicode编码，只需要在前面补0，即00000000 00101101。

由于Unicode比ASCII多占了一倍的存储空间，所以又出现了把Unicode转化为“可变长编码”的UTF-8编码。根据不同的符号而变化字节长度，当字符在ASCII码的范围时，就用一个字节表示，保留了ASCII字符一个字节的编码做为它的一部分，注意的是unicode一个中文字符占2个字节，而UTF-8一个中文字符占3个字节）。从unicode到uft-8并不是直接的对应，而是要过一些算法和规则来转换。在计算机内存中，统一使用Unicode编码，而在传输时就转换成UTF-8。

list和tuple

Python内置的一种数据类型：list是一种有序的集合，可以随时添加和删除其中元素。类似c/java中的数组

不同的是，list是可变的有序表。
增加：可以用array.append(6)追加元素到末尾；也可以用array.insert(0,0)把元素插入到指定位置。
删除：可以用array.pop()删除末尾元素；也可以用array.pop(i)其中i是索引，删除指定元素。
修改：直接赋值给对应的索引位置。
list的元素也可以是list，这样就组成了二维甚至更多维数组。

tuple：元组。与list类似，但tuple一旦初始化就不能再修改，即定长定值。
可以list作为tuple的元素实现tuple可变

循环

Python的循环有两种，一种是for…in循环，依次把list或tuple中的每个元素迭代出来.

第二种循环是while循环，只要条件满足，就不断循环，条件不满足时退出循环.
break/continue用法与c/java相同。

dict & set

Python内置了字典：dict的支持，dict全称dictionary，在其他语言中也称为map，使用键-值（key-value）存储，具有极快的查找速度。
和list比较，dict有以下几个特点：

  • 查找和插入的速度极快，不会随着key的增加而变慢；
  • 需要占用大量的内存，内存浪费多。

而list相反：

  • 查找和插入的时间随着元素的增加而增加；
  • 占用空间小，浪费内存很少。

所以，dict是用空间来换取时间的一种方法。
dict可以用在需要高速查找的很多地方，在Python代码中几乎无处不在，正确使用dict非常重要，需要牢记的第一条就是dict的key必须是不可变对象。这是因为dict根据key来计算value的存储位置，如果每次计算相同的key得出的结果不同，那dict内部就完全混乱了。这个通过key计算位置的算法称为哈希算法（Hash）。

set和dict类似，也是一组key的集合，但不存储value

十进制转十六进制

采用余数定理分解，例如将255（10）转成十六进制：
255÷16=15….15(f)
15÷16=0….15(f)
这样就计到255（10）=0xff
Python内置的hex()函数：把一个整数转换成十六进制表示的字符串
