Hexo搭建本地博客

Hexo的使用需要Node.js以及Git的支持

安装Node.js，查看安装教程。

安装Git.

安装Hexo：

在磁盘任意路径下右键Git Bash Here，如我的是F:\，依次输入下列命令:

$ npm install hexo-cli -g
$ hexo init blog
$ cd blog
$ npm install
$ hexo g
$ hexo s

最终命令行显示如下：

INFO  Start processing
INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.`

访问http://localhost:4000/，便可以看到初始的博客。
